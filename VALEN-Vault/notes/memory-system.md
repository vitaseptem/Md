# Memory System

O cérebro de longo prazo do VALEN. Armazena notas em Markdown legíveis por
humanos e IAs, indexadas em um grafo de conhecimento.

Faz parte da [[arquitetura]] geral e é consumido pelos [[agents]]. A v2 trará
busca semântica por embeddings, conforme descrito em [[roadmap]].
