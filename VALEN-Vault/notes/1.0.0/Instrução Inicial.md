# Instrução Inicial - VALEN v1.0.0 Refatoração

Vou te enviar 3 arquivos:

1. `VALEN_v1_0_0_MASTER_PROMPT_v4_ULTRA_DETALHADO.md` → Prompt mestre completo (visão + regras + arquitetura)
2. `INSTRUÇÃO OBRIGATÓRIA DE EXECUÇÃO.md` → Ordem exata das etapas que você deve seguir
3. `valen-nucleo.html` → Referência visual para o núcleo neural (use como base para melhorar a interface)

- Você **deve seguir rigorosamente** a ordem de etapas descrita no arquivo `INSTRUÇÃO OBRIGATÓRIA DE EXECUÇÃO.md`
- **Nunca avance para a próxima etapa** sem minha confirmação explícita
- Comece sempre pela **Etapa 0** (Análise e Plano)
- Após terminar cada etapa importante, pare, me mostre o que foi feito e os próximos passos
- Priorize estabilidade e simplicidade na v1.0.0. Não tente implementar tudo que está no prompt de uma vez.
