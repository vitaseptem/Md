PADRÃO OBRIGATÓRIO PARA TODAS AS TOOLS

Toda Tool deverá herdar de uma interface base.

Exemplo:

```cpp
#pragma once

#include <string>

class ITool {

public:

virtual ~ITool() = default;

virtual std::string getName()=0;

virtual std::string getDescription()=0;

virtual bool initialize()=0;

virtual bool execute(
std::string input
)=0;

virtual void shutdown()=0;

};

Nenhuma Tool deve funcionar isoladamente.

Toda Tool obrigatoriamente:

registrar eventos

gerar logs

responder métricas

informar uso de recursos

informar status

possuir permissões

possuir prioridades

possuir histórico

REGRAS ABSOLUTAS DE IMPLEMENTAÇÃO

1. Nunca criar código monolítico


2. Nunca criar dependências circulares


3. Nunca conectar ferramentas diretamente


4. Toda comunicação passa por NeuroValen


5. Usar interfaces


6. Usar eventos


7. Usar injeção de dependência


8. Usar smart pointers


9. Usar logs estruturados


10. Usar paralelismo quando necessário


11. Criar testes automatizados


12. Preparar expansão futura


13. Toda nova Tool deve seguir exatamente este padrão
